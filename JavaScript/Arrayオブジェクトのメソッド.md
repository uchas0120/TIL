## 要素を追加、削除するメソッド(push/pop/unshift/shift/splice)
```javascript
const nums = [1, 2, 3];

// push()：末尾に追加する。
nums.push(4);
nums; //  [1, 2, 3, 4]

// pop()：末尾を取り除く。
nums.pop();
nums; //  [1, 2, 3]

// unshift()：先頭に追加する。
nums.unshift(0);
nums; //  [0, 1, 2, 3, 4]

// shift()：先頭を取り除く。
nums.unshift();
nums; //  [1, 2, 3, 4]
```
