変数の宣言に使用するvar, let, constだが、それぞれ性質が違うため、上手く使い分けるようになる必要がある。  
それぞれの違い  
|          | var          | let              | const            | 
| :--------: | :------------: |  :------------:  |  :------------:  | 
| 再宣言   | 〇           | ✖               | ✖               | 
| 再代入   | 〇           | 〇               | ✖               | 
| スコープ | 関数スコープ | ブロックスコープ | ブロックスコープ | 
| 巻き上げ | undefined    | エラー           | エラー           |
