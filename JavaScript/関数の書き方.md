## 通常時
```function 関数名 (パラメータ) {
　実行したい処理
}```
```javascript
const sum = function numbers(num1, num2) {
    return num1 + num2;
} 
sum(5,6); // 11
```

## 無名関数
変数の中で関数を定義する場合、関数名を定義しなくてもよい。  
呼び出す際は変数名を記述するため、関数名がなくても問題ない。
```javascript
const sum = function (num1, num2) {
    return num1 + num2;
} 
sum(5,6); // 11
```
